### 1. Complexity Analysis (`COMPLEXITY.py`)
Understanding how to measure code efficiency.
- **Time Complexity**: Big-O notation (O(1), O(n), O(log n), etc.).
- **Space Complexity**: Memory usage explanations.
- **Real-life Analogies**: Simple explanations to grasp abstract concepts.

### 2. Recursion & Backtracking (`Recursion_backtracking.py`)
Mastering the art of functions calling themselves and exploring possibilities.
- **Recursion Basics**: Base cases, recursive steps, and factorial examples.
- **Backtracking Template**: A standard template to solve backtracking problems.
- **Key Problems**:
  - Subsets
  - Permutations
  - Printing 1 to N

### 3. String Manipulation (`string.py`)
Essential string operations and algorithms.
- **Basics**: Traversal, Immutability, and String Methods.
- **Common Algorithms**:
  - String Reversal & Palindrome Checks
  - Anagram Checks
  - Frequency Counting
  - Two Pointer Technique
- **Advanced**: Longest Substring Without Repeating Characters (Sliding Window).
